# Objective 行事易

也许还有收集箱、项目等 GTD 的功能, 正在被慢慢的简化和抛弃.

## Android Snapshot
**Android 版本还在适配调试中, 部分界面和功能会有错乱, 近期会更新.**


### Install

```
npm install
rnpm link
react-native run-ios
react-native run-android
```

iOS 还需要把ART和Notification在xcode中配置.
